# Tic‑Tac‑Toe

A classic two‑player Tic‑Tac‑Toe game built with vanilla HTML, CSS, and JavaScript.  
The game uses the module pattern (IIFEs) to keep the code organised and free of global variables.

## Features

- Two‑player gameplay – Players can enter custom names.
- Clean, responsive interface – Works on desktop and mobile.
- Turn indicator – Shows which player’s turn it is.
- Win / Tie detection – Automatically announces the winner or a tie.
- Restart button – Resets the game without refreshing the page.
- Modular code – Uses the module pattern to keep the code maintainable and scalable.

## How to Play

1. Enter names for Player X and Player O (or leave the defaults).
2. Click **Start Game** to begin.
3. Players take turns clicking on any empty square.
4. The game ends when a player gets three in a row (horizontal, vertical, or diagonal) or the board is full (tie).
5. Press **Restart** to start a new game with the same players.

## Technologies Used

- **HTML5** – Structure and semantic elements.
- **CSS3** – Grid layout, flexbox, and responsive design.
- **JavaScript (ES6)** – Core game logic, DOM manipulation, and module pattern.
